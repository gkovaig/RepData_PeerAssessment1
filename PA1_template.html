Calculate and report the total number of missing values in the dataset (i.e.the total number of rows with NAs)

goodObs = sum(complete.cases(activity))

There are 15264 observations which are not missing, out of a total of 17568 rows (86.8852%).

Strategy for filling in all of the missing values in the dataset: Will use mean for that 5-minute interval from all days in the dataset. If all the values for a certain interval is NA, then use 0.

# Create a new dataset that is equal to the original dataset but with the missing data filled in.
activity.1 = merge(activity, activityByInterval, by = c('interval'), all.x = TRUE)
activity.2 = activity.1[is.na(activity.1$steps)]
activity.2$steps = activity.2$stepsMean
activity.3 = rbind(activity.1[!is.na(activity.1$steps)], activity.2)
